# Icons 应用统一设计规范文档

## 1. 设计理念
本设计规范旨在为 Icons 应用建立一套统一、现代化的界面设计标准，确保所有 UI 组件在视觉风格、交互方式和用户体验上保持一致性。

## 2. 色彩系统

### 2.1 主要颜色
- 主色：蓝色 (#007AFF)
- 辅助色：紫色 (#5856D6)
- 成功色：绿色 (#34C759)
- 警告色：橙色 (#FF9500)
- 错误色：红色 (#FF3B30)

### 2.2 中性色
#### 浅色主题
- 背景色：#F2F2F7
- 表面色：#FFFFFF
- 主要文字：#000000
- 次要文字：#8E8E93
- 边框色：#D1D1D6

#### 深色主题
- 背景色：#000000
- 表面色：#1C1C1E
- 主要文字：#FFFFFF
- 次要文字：#98989D
- 边框色：#2C2C2E

### 2.3 颜色使用规范
- 主色用于重要操作按钮和核心功能标识
- 辅助色用于次要操作按钮和功能标识
- 成功色用于成功状态提示和确认操作
- 警告色用于警告信息和需要注意的操作
- 错误色用于错误状态提示和危险操作

## 3. 间距系统

### 3.1 基础间距单位
- XS: 4px
- S: 8px
- M: 16px
- L: 24px
- XL: 32px
- XXL: 48px

### 3.2 间距使用规范
- 组件内元素间距使用 XS 或 S
- 组件间间距使用 M 或 L
- 页面级间距使用 L、XL 或 XXL
- 导航栏高度建议使用 44px 或 48px
- 按钮高度建议使用 36px 或 44px

## 4. 字体系统

### 4.1 字体层级
| 层级 | 字体大小 | 字体粗细 | 使用场景 |
|------|----------|----------|----------|
| 大标题 | 32px | Semibold | 页面主标题 |
| 标题1 | 24px | Semibold | 章节标题 |
| 标题2 | 20px | Semibold | 小节标题 |
| 标题3 | 18px | Semibold | 卡片标题 |
| 正文 | 16px | Regular | 主要内容 |
| 辅助文字 | 14px | Regular | 描述信息 |
| 小字 | 12px | Regular | 标签、注释 |

### 4.2 字体使用规范
- 中文使用系统默认中文字体
- 英文使用 SF Pro Display 或 SF Pro Text
- 标题使用 Semibold 或 Bold 提高可读性
- 正文使用 Regular 确保阅读舒适度
- 行高建议为字体大小的 1.2-1.5 倍

## 5. 圆角系统
- 小圆角：4px（按钮、输入框等小控件）
- 中圆角：8px（卡片、弹窗等中等控件）
- 大圆角：12px（页面级组件）
- 全圆角：用于头像、徽章等圆形元素

## 6. 阴影系统

### 6.1 阴影层级
1. 高度 1 (卡片)：
   `shadowColor: #00000020, shadowOffset: (0, 1), shadowOpacity: 0.1, shadowRadius: 2`
2. 高度 2 (弹窗)：
   `shadowColor: #00000030, shadowOffset: (0, 2), shadowOpacity: 0.15, shadowRadius: 4`
3. 高度 3 (浮动按钮)：
   `shadowColor: #00000040, shadowOffset: (0, 4), shadowOpacity: 0.2, shadowRadius: 8`

### 6.2 阴影使用规范
- 卡片元素使用高度 1 阴影
- 弹窗、下拉菜单使用高度 2 阴影
- 浮动按钮、悬浮层使用高度 3 阴影
- 深色主题下可适当增强阴影效果

## 7. 交互动效

### 7.1 动画时长
- 快速反馈：150ms
- 状态变化：250ms
- 页面切换：300ms
- 复杂动画：500ms

### 7.2 动画曲线
- 标准缓动：easeInOut (cubic-bezier(0.4, 0, 0.2, 1))
- 进入动画：easeOut (cubic-bezier(0, 0, 0.2, 1))
- 退出动画：easeIn (cubic-bezier(0.4, 0, 1, 1))

### 7.3 交互反馈
- 按钮点击：按压缩放 0.95 倍
- 开关切换：滑动位移 + 颜色变化
- 加载状态：旋转动画
- 错误提示：震动动画 + 颜色变化

## 8. 响应式布局

### 8.1 断点设置
- 小屏幕：≤ 768px
- 中等屏幕：769px - 1024px
- 大屏幕：≥ 1025px

### 8.2 布局规范
- 侧边栏宽度可调节范围：200px - 400px
- 主内容区域最小宽度：400px
- 卡片网格列数根据屏幕宽度自适应（2-6列）
- 表单元素在小屏幕下垂直堆叠排列

## 9. 可访问性

### 9.1 颜色对比度
- 文字与背景对比度至少达到 4.5:1
- 重要交互元素与背景对比度至少达到 3:1
- 禁用状态元素与背景对比度至少达到 2:1

### 9.2 键盘导航
- 所有交互元素支持 Tab 键导航
- 按钮和链接支持 Enter 键激活
- 表单元素支持标准键盘操作
- 提供清晰的焦点指示器

### 9.3 屏幕阅读器
- 为图标提供适当的替代文本
- 为表单元素提供标签关联
- 为动态内容变化提供通知
- 遵循 ARIA 标准实现语义化标记

## 10. 组件规范

### 10.1 按钮
- 高度：36px 或 44px
- 内边距：水平 16px-24px
- 圆角：4px 或 8px
- 文字：Semibold，居中对齐

### 10.2 输入框
- 高度：36px 或 44px
- 内边距：左侧 12px，右侧 8px
- 边框：1px 实线
- 圆角：4px 或 8px

### 10.3 卡片
- 内边距：16px 或 24px
- 圆角：8px 或 12px
- 阴影：高度 1
- 边框：可选

### 10.4 导航栏
- 高度：44px 或 48px
- 项目间距：16px
- 激活指示器：2-3px 高度下划线
- 文字：Semibold

## 11. 图标和图像

### 11.1 图标规范
- 尺寸：16px, 20px, 24px, 32px
- 线条粗细：2px 或 2.5px
- 颜色继承父级文字颜色
- 提供 24px 作为默认尺寸

### 11.2 图像规范
- 优先使用 2x 和 3x 高清资源
- 图像比例保持一致（1:1, 4:3, 16:9）
- 图像容器统一圆角：8px
- 提供占位符和加载状态
